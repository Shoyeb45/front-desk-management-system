## Custom classes

```
// 🎨 Backgrounds (auto dark mode)
bg-background              // Main page background
bg-background-secondary    // Cards, panels
bg-background-tertiary     // Hover states, table headers

// 📝 Text (auto dark mode)
text-foreground           // Main text
text-foreground-secondary // Labels, secondary text
text-foreground-muted     // Placeholders, disabled

// 🔲 Borders (auto dark mode)
border-border             // Standard borders
border-border-focus       // Focus rings
border-border-divider     // Divider lines

// 🎯 Brand Colors (same in both modes)
bg-primary-500 / text-primary-500     // Main brand color
bg-secondary-500 / text-secondary-500 // Secondary brand

// ⚡ Status Colors (same in both modes)
bg-success-500    // Green
bg-warning-500    // Orange
bg-danger-500     // Red
bg-info-500       // Blue
```
